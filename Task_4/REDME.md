# Отчет по четвертой лабе

## 1. One-Sample t-Test — Одновыборочный t-тест

- Компания утверждает: среднее время доставки = 30 минут.
- Вы взяли выборку из 15 наблюдений → проверяем, отличается ли реальное среднее от 30.

```bash
1. ONE-SAMPLE t-TEST
t-statistic: 0.6756639246921738  p-value: 0.5102612860539539
```

- Если p < 0.05 → доставка действительно отличается от 30 минут.
- Если p > 0.05 → нет доказательств отклонения.

## 2. Two-Sample Independent t-Test — Двувыборочный независимый t-тест

Сравниваем тестовые баллы:
- традиционный метод обучения
- новый метод

```bash
2. TWO-SAMPLE INDEPENDENT t-TEST
t-statistic: -3.0660225761688804  p-value: 0.007878599579979169
```

- p < 0.05 → методы дают разные результаты.
- В нашем случае новый метод значимо лучше.

## 3. Paired t-Test — Парный t-тест

- Вес до диеты и после диеты у 12 участников.

```bash
3. PAIRED t-TEST
t-statistic: 6.750661343263234  p-value: 3.154609097315341e-05
```

- p < 0.05 → диета действительно влияет на вес.
- В нашем случае — уменьшает.

## 4. One-Sample t-Test with Normally Distributed Data

- Проверяем, отличается ли средний игровой балл от 500.

```bash
4. ONE-SAMPLE t-TEST WITH NORMAL DATA
t-statistic: 2.2044551627605733  p-value: 0.03558027071269493
```

- Обычно p > 0.05 → средний результат не отличается от 500.

## 5. One-Sample z-Test — Одновыборочный z-тест для среднего

- Заявлено: средняя длительность батареи = 10 часов.
- В выборке из 50 телефонов получилось: 9.5 часов.

```bash
5. ONE-SAMPLE z-TEST (MEAN)
z-statistic: -2.946278254943948  p-value: 0.0032162293101274386
```

- p < 0.05 → средняя длительность батареи действительно меньше 10.

## 6. Two-Sample z-Test — Двувыборочный z-тест для средних

Средние баллы учеников:
- школа A → 75
- школа B → 78

```bash
6. TWO-SAMPLE z-TEST (MEANS)
z-statistic: -2.1821789023599236  p-value: 0.029096331741252257
```

- p < 0.05 → различия значимы.

## 7. One-Sample z-Test for Proportion — Одновыборочный z-тест для долей

- Утверждают: 60% предпочитают онлайн-покупки.
- Вы нашли: 58%.

```bash
7. ONE-SAMPLE z-TEST (PROPORTION)
z-statistic: -0.9128709291752777  p-value: 0.36131042852617834
```

- p > 0.05 → разница статистически незначима.

## 8. Two-Sample z-Test for Proportions — Двувыборочный z-тест для долей

Пользуются общественным транспортом:
- Город X: 25%
- Город Y: 26.67%

```bash
8. TWO-SAMPLE z-TEST (PROPORTIONS)
z-statistic: -0.8884223652872202  p-value: 0.3743135992557667
```

- p < 0.05 → пропорции действительно различаются.

## 9. ANOVA — Дисперсионный анализ

Весовой прирост детей:
- только грудное вскармливание
- только смесь
- смесь + грудное

```bash
9. ANOVA TEST
F-statistic: 26.701251857537958  p-value: 7.18623550288582e-09
```

- p < 0.05 → хотя бы одна группа отличается.
- Но ANOVA не говорит какая именно.

## 10. Chi-Squared Test — χ²-тест независимости

- зависит ли риск-аппетит от пола?

```bash
10. CHI-SQUARED TEST
Chi-sq statistic: 7.094218206100441  p-value: 0.13099220858734012

Expected Frequencies:
[[ 43.21212121  24.74242424  28.22727273  32.40909091 101.40909091]
 [ 80.78787879  46.25757576  52.77272727  60.59090909 189.59090909]]
```

- p < 0.01 → зависимости есть
- p > 0.01 → пол и риск-аппетит независимы
